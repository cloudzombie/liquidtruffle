# bare-hyperevm

Minimal HyperEVM starter generated by `liquidtruffle`.

## Scripts

```bash
npm install
npm run compile
npm run test
npm run doctor
npm run deploy:testnet
npm run deploy:mainnet
```

## Notes

- Mainnet chain ID: `999`
- Testnet chain ID: `998`
- Mainnet RPC default: `https://rpc.hyperliquid.xyz/evm`
- Testnet RPC default: `https://rpc.hyperliquid-testnet.xyz/evm`
- Built-in LiquidTruffle testnet fallback: `https://998.rpc.thirdweb.com`
- Set `PRIVATE_KEY` in `.env` before deploying or sending write transactions.
- Successful deploys are written to `deployments/<network>.json` for GUI inspection.
- `deploy:testnet` probes the official HyperEVM testnet RPC first and falls back explicitly if it is unreachable.

If deployment gas is above the default 2M core block limit, follow the Hyperliquid big-block flow before retrying deployment.
